# Run the solutions
Change into directory that contains the solution to run.

## C#
`dotnet test solution.sln`

## Java
`java Solution.java`

## JavaScript
`node solution.js`

## Perl
`perl solution.pl`

## Swift
`swift solution.swift`

## TypeScript
`deno run --allow-read solution.ts`
