# Run the solutions
Change into directory that contains the solution to run.

## Java
`java Solution.java`

## JavaScript
`node solution.js`

## Perl
`perl solution.pl`

## Swift
`swift solution.swift`

## TypeScript
`tsc solution.ts && node solution.js`
